Tools

Hypertext Markup Language (HTML) is used to create web pages. Most of the writing seen on the web page is due to HTML. It is like the backbone of the web page. I used the div element to divide up the page into sections full of headings, paragraphs, inputs and tables.

Cascading Style Sheet (CSS) is used to style the web pages. I used a separate file for the stylesheet. I used ids to style individual elements and class names to style groups of elements. I styled the navigation panel and headers to have the same theme throughout the pages.

JavaScript is used to make the web page interactive. Anything clicked on the page used a javascript function to do its task. I put the javascript at the bottom of the body section but I could have had a whole new file. Javascript allowed me to enter the inputs into the table and add a new category. It also helped open up the lightbox and the slides in them.
